Track the major updates and improvements to the Family Group Record Builder

v0.0.1 September 2025
Initial release of the Family Group Record Builder with comprehensive functionality for creating professional genealogy records.
  • New Complete family group record form with father and mother sections
  • New Parent information fields for both parents
  • New Dynamic children management with add/remove functionality
  • New Event management system for birth, death, and marriage records
  • New Rich text editor for comments and notes
  • New Google authentication for secure user accounts
  • New Cloud storage integration with Firebase Firestore
  • New Real-time auto-save functionality to prevent data loss
  • New Preparer information section
  • New Modern, responsive design with top navigation
  • New Changelog page for tracking updates

That's all for now! Check back regularly for updates and new features.
